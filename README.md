# MPEI Registration Bot

Telegram-бот для записи на мероприятия в МЭИ. Работает через **Webhook**, серверная часть на **Flask**, для публикации используется **ngrok**.

## 🚀 Возможности

*   Регистрация пользователей (имя, фамилия, группа).
*   Просмотр ближайших мероприятий.
*   Запись и отмена записи на мероприятия.
*   Автоматический контроль свободных мест.
*   Административная панель (добавление/удаление мероприятий, просмотр списка участников, добавление других администраторов).
*   Правила аренды площадки МЭИ.
*   Проверка корректности ввода данных (имя, группа, дата, время).

## 🛠 Технологии

*   **Python 3** – основной язык.
*   **pyTelegramBotAPI** – библиотека для работы с Telegram Bot API.
*   **Flask** – веб-фреймворк для обработки webhook-запросов.
*   **SQLite** – база данных (управление через `DbHelper` в `data.py`).
*   **ngrok** – для создания публичного HTTPS-туннеля к локальному серверу.

## 🔐 Права и роли

- **Обычный пользователь:** регистрация, просмотр мероприятий, запись/отписка.

- **Администратор:** имеет расширенное меню:
  - Добавление/удаление мероприятий.
  - Просмотр списка записавшихся.
  - Добавление новых администраторов (по пересланному сообщению или Telegram ID).
